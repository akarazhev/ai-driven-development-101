AI-Driven Development Course

A practical, production-minded curriculum on using AI to design, build, and ship software. The course is written in Markdown and organized by chapters, one folder per chapter.

  • Technology-agnostic: Learn prompt engineering and AI-assisted development patterns applicable to any technology stack
  • Format: Text-first with images; each chapter has its own README.md
  • Goal: Enable developers to work as senior full-stack developers using AI assistance, regardless of their technology stack

Authors

  • Alexey Mikhalchenkov
  • Klim Izmaikov
  • Andrey Karazhev

Syllabus

  1. 00 - Rules and Prompt Optimization - Create and manage Cursor rules to simplify prompts
  2. 01 - Introduction to AI - Fundamentals of AI-assisted development
  3. 02 - When to Use AI for Complex Tasks - Breaking down complex tasks and multi-step workflows
  4. 03 - Setup Cursor AI - Configure Cursor for effective AI pair programming
  5. 04 - Optional: Enhanced Context with contex 7 - ⚠️ Optional: Only if you need enhanced context for large codebases
  6. 05 - AI-Driven Software Development - Practical workflows and patterns
  7. 06 - Project: Confluence Publisher App - Full-stack project implementation

How to use this course

  • Start at chapter 01 and proceed in order.
  • Each chapter provides learning objectives, an outline, and hands-on exercises.
  • Keep assets (images) within the respective chapter folder.
