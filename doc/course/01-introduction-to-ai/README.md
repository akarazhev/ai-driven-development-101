# 01. Introduction to AI

A concise introduction to modern AI concepts relevant to software development.

## Learning objectives

- Understand how LLMs work at a high level (tokens, context window, prompting)
- Recognize appropriate use-cases and limitations for AI in software engineering
- Establish responsible use guidelines (privacy, security, licensing)

## Prerequisites

- Python 3.13 installed
- Basic Git and terminal usage

## Outline

- LLM fundamentals and key terminology
- Prompting basics and evaluation practices
- Safety, privacy, and responsible AI usage

## Exercises

- Draft and iterate on a prompt for a coding task
- Compare outputs and document observations

## References

- Official Python docs
- Responsible AI guidelines (org-specific if applicable)
