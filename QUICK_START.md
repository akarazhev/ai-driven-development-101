# Быстрый старт

## Установка

```bash
pip install requests
```

## Запуск

```bash
python check_confluence_posting.py
```

## Что делает скрипт?

1. ✅ Проверяет подключение к Confluence
2. ✅ Проверяет аутентификацию (username + API token)
3. ✅ Проверяет доступ к пространству SPGAC
4. ✅ Проверяет права на чтение
5. ✅ **Создает тестовую страницу** (проверяет права на запись)
6. ✅ Удаляет тестовую страницу (автоматически)

## Результат

Если все шаги пройдены успешно, вы увидите:
```
✓ ALL TESTS PASSED: Can post to Confluence!
```

## Если что-то пошло не так

### Ошибка подключения
- Проверьте доступность сервера: `https://pmc-stage.specific-group.eu/confluence`

### Ошибка аутентификации (401)
- Проверьте правильность username и API token

### Ошибка прав доступа (403)
- Убедитесь, что пользователь `spg.academy` имеет права на создание страниц в пространстве SPGAC

### Страница не создается
- Проверьте права пользователя в Confluence
- Убедитесь, что пространство SPGAC существует и доступно

## Использование переменных окружения (рекомендуется)

Для безопасности можно использовать переменные окружения:

```bash
export CONFLUENCE_API_TOKEN="your_token_here"
python check_confluence_posting.py
```

## Дополнительные опции

В коде можно изменить параметры:

```python
# Оставить тестовую страницу (не удалять)
success = tester.run_full_test(space_key="SPGAC", cleanup=False)

# Выполнить только отдельные шаги
tester.test_connection()
tester.test_authentication()
```
